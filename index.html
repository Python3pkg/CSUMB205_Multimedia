CSUMB's CST205 - A simple Image Class

This project offers students acces to a simple RGBImage class, which is built on the use of the Image module in the PIL package. As PIL is no longer updated for Python 3, this package relies on Pillow.

This package will assume that the user is using at least Python 3.4

You can download this package through the use of PIP. Make sure that you have "pip" installed (comes with an install of Python 3.4.2), and from the command line, run

pip install CSUMB205-Image

On Mac OSX, you will have multiple versions of Python. You still need to install Python 3.4.2, but when you run pip, you must use

pip3 install CSUMB205-Image
