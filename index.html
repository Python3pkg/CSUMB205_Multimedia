CSUMB's CST205 - A simple Image Class

This project offers students access to a simple rgbImage class, which is built on the use of the Image module in the PIL package. As PIL is no longer updated for Python 3, this package relies on Pillow.

This package assumes that the user is using at least Python 3.4.2.

If you have not installed Python 3.4.2, please download it from python.org.

If you are using Windows, make sure to check the option to add the python.exe to your path. PIP will not work otherwise.

INSTALLATION

You can download this package through the use of PIP. From an administrator command line, run...

 pip install CSUMB205-Image

On Mac OSX, you will have multiple versions of Python. You still need to install Python 3.4.2, but when you run pip, you must use...

 pip3 install CSUMB205-Image

USAGE:

The interface is meant to be as simple as possible. This is how you import the necessary class.

 from simpleImage.ImageClass import rgbImage

This is how you actually create an Image object.

 myImage = rgbImage("/absolute/path/to/your/image")

NOTE: If you leave the constructor empty, tkinter will open a file dialog, asking you to choose a file.

 myImage = rgbImage("")  ===>  tkinter filedialog GUI

Now that we have an image imported, what values do we have? We can get the image's size, or number of pixels.

 numOfPixels = myImage.size

We can also get the width and heigh individually.

 myWidth = myImage.width
 myHeight = myImage.height

The most important part of our image, however, is the list of pixels. We can get individual pixels by using two array references, where the first value is the row of picture (starting from 0), and the second is the column of the picture (starting from 0).

 a_pixel = myImage.pixels[0][0]

Each element of the images list "pixels" is an rgbPixel object. These objects have a "red", "green", and "blue" attribute, which are integers. We can then loop through every pixels and set it's blue or red values.

  for row in myImage.pixels:
      for pixel in row:
          pixel.setRed(255)
          pixel.setBlue(0)

If a pixel's attribute is set to anything above 255, the object will automatically wrap the value around. For example, setting a pixel's red value to 300 will actually set it to 300-256, or 44.

 myImage.pixels[0][0].setRed(300)  ===>  myImage.pixels[0][0] red is now 44

For more information on the rgbImage class, type help('SimpleImage.ImageClass') in the Python interpreter
